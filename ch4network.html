  1. Network Classifications
    • LAN

      A LAN is a network that connects devices within a limited geographic area, like a home, office, or campus.

    • MAN

      A MAN covers a larger area, like a city, connecting multiple LANs across a metropolitan region.

    • WAN

      A WAN spans a large geographic area, connecting LANs and MANs across long distances.

  2. Network Models
    • Bus Network

      A bus network uses a single central cable to connect all devices; data is transmitted to all devices on the network.

    • Star Network

      In a star network, all devices are connected to a central hub; data is transmitted through the hub.

  3. Protocols
    • CSMA/CD Ethernet

      CSMA/CD is a protocol used in Ethernet networks to manage access to the network medium, detect and resolve collisions.

    • CSMA/CA WiFi

      CSMA/CA is a protocol used in Wi-Fi networks to manage access to the wireless medium, avoiding collisions.

  4. Combining Networks
    • Repeater

      A repeater is a network device that amplifies and retransmits network signals to extend the range of a network.

    • Bridge

      A bridge connects two or more network segments and filters traffic between them based on MAC addresses.

    • Switch

      A switch is a network device that forwards data only to the device that needs it within a local network.

    • Router

      A router connects different networks and directs data between them based on IP addresses.

    • Gateway

      A gateway is a device or software that connects different network protocols or networks, facilitating communication between them.

  5. Client/Server Model
    • P2P - Peer-to-Peer

      P2P is a decentralized network model where devices communicate directly with each other without a central server.

    • Distributed System

      A distributed system is a network of interconnected computers that work together to achieve a common goal.

    • Cloud Computing

      Cloud computing refers to the delivery of computing services (e.g., storage, processing) over the internet from remote data centers.

  6. Internet Addressing
    • Tier 1

      Tier 1 networks are high-level internet service providers that connect directly to the global internet infrastructure.

    • Tier 2

      Tier 2 networks connect to Tier 1 networks and serve as regional or national providers.

    • Tier 3

      Tier 3 networks are local and connect to Tier 2 networks, providing internet access to end-users.

  7. IP Address
    • ICANN

      ICANN is a nonprofit organization responsible for managing the allocation of IP addresses and domain names.

    • DNS

      DNS is a system that translates human-readable domain names into IP addresses, enabling internet communication.

    • TLD

      A TLD is the highest level in the domain naming system (e.g., .com, .org, .net).

    • Country-code TLDs

      Country-code TLDs are domain extensions associated with specific countries or territories (e.g., .uk, .fr).

  8. Internet protocols
    • FTP

      FTP is a protocol for transferring files between a client and a server on a network, often used for website maintenance.

    • HTTP

      HTTP is the protocol used for transmitting web pages, providing a foundation for the World Wide Web.

    • SMTP

      SMTP is used for sending email messages between servers, enabling email communication.

  9. World Wide Web
    • URL

      A URL is a web address that specifies the location of a resource on the internet.

    • Client-side activities

      Client-side activities refer to processes that occur on the user's device, such as rendering web pages in a browser.

    • Server-side activities

      Server-side activities are processes that occur on the web server, handling requests and serving dynamic content.

  10. Internet four-layer software
    • Application layer

      The application layer is the top layer in the OSI model and is responsible for end-user services and applications.

    • Transport layer

      The transport layer manages data transfer, ensuring reliable communication between devices.

    • Network layer

      The network layer handles routing and forwarding of data packets between devices in different networks.

    • Link layer

      The link layer manages the physical connection and data framing between devices on the same network segment.

  11. Internet is:
    • Packet-switching network

      The internet uses packet-switching, breaking data into packets for efficient transmission and reassembly at the destination.

    • Fault-tolerant

      The internet is designed to withstand failures and continue functioning in the presence of faults or errors.

    • Redundant

      The internet is built with redundancy, ensuring alternative paths for data in case of network disruptions.

  12. Cybersecurity
    • Malware

      Malware refers to malicious software designed to harm or compromise computer systems and data.

    • Virus

      A virus is a type of malware that can replicate itself and spread to other files and systems.

    • Worm

      A worm is a self-replicating malware that can spread independently over a network or the internet.

    • Trojan horse

      A Trojan horse is malware disguised as legitimate software, tricking users into installing it.

    • Spyware

      Spyware is software that secretly gathers user information or monitors activities without consent.

    • Phishing

      Phishing is a cyber-attack that tricks individuals into revealing sensitive information or credentials.

    • DDoS attack - Distributed Denial of Service

      A DDoS attack floods a server with traffic, overwhelming it and causing service disruptions.

  13. Protection
    • Firewall

      A firewall is a network security device that filters incoming and outgoing network traffic based on predetermined security rules.

    • Spam filter

      A spam filter is a software or system that identifies and blocks unsolicited email, known as spam.

    • Proxy server

      A proxy server acts as an intermediary between client devices and the internet, providing security and anonymity.

    • Antivirus software

      Antivirus software detects and removes malware and viruses from computer systems, enhancing security.

    • Cryptography

      Cryptography involves the use of codes and encryption to secure data and protect it from unauthorized access.

    • Public key encryption

      Public key encryption is a method for securing data by using public and private keys for encryption and decryption.
