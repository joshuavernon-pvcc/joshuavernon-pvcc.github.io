Term Definition
operating system Software that supports a computer's basic functions.
job queue A list of tasks to be carried out in turn by an OS.
FIFO A method where the job queue is first in first out.
interactive processing A person needs to provide instructions to the computer while it's processing.
time-sharing Where a computer resource is shared among tasks or users for multi-tasking.
multiprogramming Multiple programs are running on a single processor.
application software Performs a specific task other than the one relating to the operation of the computer, like Word processors or media players.
system software Runs a computer's hardware and application programs.
utility software Helps users manage, mantain, and optimize their computer systems.
graphical user interface (GUI) Allows users to interact through graphical icons instead of text-based UIs.
shell Exposes an OS's services to the user or another program.
kernel Connects and manages resources for a computer's OS and components.
device driver A program that operates or controls a specific device that's attached to the computer.
file manager Provides user interface to manage files and folders.
memory manager Controls and coordinates a computer's main memory.
virtual memory Uses hardware and software to compensate for physical memory shortages by temporarily transferring data from RAM to disk storage.
boot strapping Process of starting a computer.
ROM Read-only memory that is non-volatile and can't be electronically modified.
deadlock Where two programs that share the same resource prevent each other from accessing it.
spooling Data is temporarily held so it can be used and executed by a device, program, or the system.
super user (or administrator) A user with special privileges that are needed to maintain the system.
multi-factor authentication The user needs to provide two or more verification factors to log in or gain access to a device.
